# Tokyo Ramen House

**Modern Japanese Ramen Restaurant Landing Page**

Website landing page restoran ramen Jepang yang modern, stylish, dan menggugah selera, terinspirasi dari ramen shop autentik di Tokyo. Dibangun menggunakan **HTML5, CSS3, dan Vanilla JavaScript** dengan fokus pada UI/UX yang clean, responsive, dan premium.

---

##  Features

*  Hero section dengan visual ramen yang menarik
*  Fully responsive (mobile, tablet, desktop)
*  Sticky navigation bar
*  Menu cards dengan hover effect
*  Gallery section dengan grid layout
*  Testimonials section
*  Location & reservation section
*  Mobile hamburger menu
*  Smooth scrolling navigation
*  Modern Japanese aesthetic design

---

##  Technologies Used

* **HTML5** – Semantic structure
* **CSS3** – Styling & layout
* **Vanilla JavaScript** – Interactivity
* **Flexbox & CSS Grid** – Layout system
* **Media Queries** – Responsive design

##  How to Run Locally

1. Clone repository ini:

```bash
git clone https://github.com/ArdhikaTama/tokyo-ramen-house.git
```

2. Masuk ke folder project:

```bash
cd tokyo-ramen-house
```

3. Buka file `index.html` di browser:

```bash
double click index.html
```

Atau gunakan Live Server (VS Code extension)
